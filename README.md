## Jenkins 環境

[red-lab.tar.gz](https://drive.google.com/file/d/1nISK_8rvPYie9DAnf0OFXCdgl-4pltSi/view?usp=sharing)

### 解壓縮到 linux 環境

```shell
tar zxvf red-lab.tar.gz
```

### 運行 Jenkins

* **port**：9080
* **帳號**：admin
* **密碼**：123456

```shell
docker-compose up -d
```
